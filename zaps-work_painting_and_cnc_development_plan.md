# 房屋油漆与 CNC 成本工具开发计划

更新时间：2026-07-14

## 目标

沿着“报价单 / 成本计算器”这一条产品主线，增加两个高意图场景：住宅油漆承包商的报价，以及 CNC 加工订单的成本核算。所有新增页面遵循同一上线条件：多语言路由、可解释公式、异常输入保护、货币与物理单位真实换算、结构化数据、sitemap、`llms.txt`、与一篇操作指南互链。

## 第 4–5 周：房屋油漆报价计算器（本次开始实现）

### 用户与任务

- 独立油漆承包商：上门后需要快速判断项目是否有利润，并产出可打印的总价。
- 小型施工团队：需要把涂料、人工、准备、交通和最低出工费放进同一套公式。
- 客户沟通：需要看到范围、涂层、条件和报价，而不是一个无法解释的“每平方单价”。

### 输入与公式

- 范围：室内 / 室外、可上漆面积、涂层数、房间数、表面状态。
- 涂料：每容器覆盖面积、单价、损耗、耗材与保护成本。
- 人工：每人每小时产能、完整人工时薪、团队人数。
- 定价：额外准备费、交通、人工间接成本、目标毛利、最低工程费、税。
- 涂料容器数：`ceil(面积 × 涂层 × (1 + 损耗) ÷ 每容器覆盖面积)`。
- 人工工时：`((面积 × 涂层) ÷ 产能 + 房间准备时间) × 状态系数 × 工程类型系数`。
- 报价：`max(直接成本 ÷ (1 − 目标毛利), 最低工程费) + 税`。

### 本期交付

- `/tools/house-painting-quote` 及 8 个语言路由。
- PDF 打印、CSV 导出、移动端实时报价条。
- WebApplication、FAQ、Breadcrumb 结构化数据；工具 sitemap 和 `llms.txt` 更新。
- 英文主操作指南，并从工具页互链。

### 不做

- 不输入或推荐“市场平均单价”。
- 不把容器覆盖率假设为某个品牌或油漆体系的事实；用户自行输入。
- 不处理特殊合规、危险材料、税法或正式合同条款。

## 第 6 周：油漆定价指南与长尾页

### 内容集群

主指南：`/guides/how-to-price-house-painting-jobs`，解释量测、容器覆盖率、生产率、准备工作、毛利和最小出工费。

第一批长尾页只做有独立计算意图、能提供不同输入或案例的页面：

1. `how-to-price-interior-painting-jobs`：房间切边、家具保护、颜色变化和低干扰施工。
2. `how-to-price-exterior-painting-jobs`：外墙进出、表面准备、气候窗口和高处作业假设。
3. `paint-coverage-calculator`：围绕面积、涂层、覆盖率、损耗和容器数的轻量入口；与报价工具共享公式，不复制一个不一致的计算器。

### 发布标准

- 每页有不同的搜索意图、公式或案例，避免用地区名批量生成薄内容。
- 所有语言版本完成后才加入指南 sitemap 的国际化条目；英文主版先由工具页与 `llms.txt` 发现。
- 更新指南索引卡、`llms.txt`、guides sitemap，并为每页添加 Article / FAQ / 内链。

## 第 7–9 周：CNC 加工成本计算器

### 用户与任务

- 小型 CNC 车间或报价工程师：从图纸 / CAM 估算快速确认单件、批量、设置和换刀后的真实成本。
- 采购或外协业务：比较不同材料、批量和加工时间下的可持续报价，而不是只看机时。

### MVP 输入

- 工艺：3 轴铣削（先不做车铣复合、5 轴和复杂工序路由）。
- 材料：材料单价、毛坯长宽厚、余量 / 损耗、可用率。
- 机台：设置时间、编程时间、单件循环时间、机器小时费率、人工时薪、装夹 / 检验时间。
- 工具与外协：刀具损耗、夹具或外协固定费、后处理、包装、运输。
- 订单：数量、目标毛利、最低订单费、税。

### 计算框架

- 材料成本 = 毛坯体积 × 材料单价 × 损耗 / 可用率。
- 总工时 = 编程 + 设置 + 数量 ×（循环 + 装夹 / 检验）。
- 直接成本 = 材料 + 机器时间 + 人工 + 刀具 + 固定加工费 + 后处理 / 包装。
- 报价 = `max(直接成本 ÷ (1 − 目标毛利), 最低订单费) + 税`；同时显示单件价与订单总价。

### 分阶段交付

1. 第 7 周：公式、数据结构、单位换算、边界测试；只支持 3 轴铣削。
2. 第 8 周：报价 UI、批量分摊、PDF / CSV、多语言、移动端审计。
3. 第 9 周：CNC 定价指南、首批长尾内容（设置时间、机时成本、批量报价），SEO 与发布复核。

### 关键边界

- 不能把用户输入的 CAM 时间包装成“自动工艺报价”。
- 材料数据库、工艺参数推荐和地域市场价格是后续增强项，不进入 MVP。
- 尺寸、体积、货币、单件 / 批量分摊必须真实换算并可测试。
